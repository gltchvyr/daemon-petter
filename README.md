# daemon-petter

A tiny browser toy where you pet a glitch daemon.

## Features
- Pet different zones: head, horns, heart, paws
- Offer cookies, trinkets, and heart sigils
- Trigger zoomies, naps, and tarot pulls
- Unlock forms, mood shifts, and small secrets
- Accidentally adopt a digital cryptid
